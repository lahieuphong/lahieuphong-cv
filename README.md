# CV Cá Nhân - La Hiểu Phong

Website CV cá nhân được xây dựng bằng **Next.js**, trình bày thông tin cá nhân, kỹ năng, kinh nghiệm, học vấn và công bố khoa học.

## Công Nghệ

Next.js, React, TypeScript, CSS, GitHub Actions, GitHub Pages.

## Chạy Local

```bash
yarn install
yarn dev
```

Mở tại:

```text
http://localhost:3000
```

## Build Và Deploy

```bash
yarn build
```

Dự án được static export và deploy tự động lên GitHub Pages thông qua `.github/workflows/deploy.yml`.
